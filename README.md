# BBDownG
BBDown的图形化版本 - 哔哩哔哩(B站)视频下载、音频下载、字幕下载 - bilibili video download

## 特性
- 可以根据用户的文本中的url循环下载

## 使用方法
1. 将BBDownG、aria2c.exe、BBDown.exe、ffmpeg.exe放在一起即可使用，或者自己在程序中指定路径
2. 其它使用方法自行摸索
